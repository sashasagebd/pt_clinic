This is a work-in-progress project for a local pt clinic to allow them to send intern information to an internship company. To run, install dependencies and set up the gmail api (put credentials in credentials.ts in the pt_electron/gmail folder). Next, change the destination email in oauth to whoever you want to send the emails to. After making all code changes, build the app, and run the executable (either via the installer or the unpacked folder exe). After adding employees and attaching images to them, select all those that need to be emailed out then press the send emails button. Currently, this app is only set up for gmail accounts via Google OAuth.
